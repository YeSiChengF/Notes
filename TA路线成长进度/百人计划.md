# 图形部分

## 第一章 基础夯实

### 渲染流水线

图形渲染管线是实时渲染的核心组件

渲染管线称为管线，因为渲染流程是线性的

整体流程:

1. 应用阶段

   准备对象的基本数据：场景里的物体、位置朝向、大小以及物体对应模型里的每个顶点位置、法线、切线等。场景光源、摄像机朝向

   - 粗粒度剔除
   - 进行渲染设置
   - 准备基本数据
   - 输出到几何阶段

2. 几何阶段

   - 顶点着色器

   - 曲面细分

   - 几何着色器

   - 顶点裁剪

     去掉看不到的位置

   - 屏幕映射

     3D坐标转换到2D坐标

3. 光栅化阶段

   - 三角形(点/线)设置
   - 三角形(点/线)遍历
   - 片段着色器

4. 逐片元操作

   - 裁剪测试
   - 透明度测试
   - 深度测试
   - 模板测试
   - 混合测试

5. 后处理
